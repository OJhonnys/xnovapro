# Xnova Pro — High-Performance Live Entertainment Landing Page

An optimized, responsive, cinematic landing page developed for **Xnova Productions Corp** (Miami, FL). This platform showcases large-scale event rosters, advanced stage design architectural builds, immersive scenography, and full-scale entertainment event organization services.

Built following modern UI/UX workflows, the system uses clean semantic markup, pure utility-first CSS layout trees, and an asynchronous data-driven JavaScript architecture.

---

## Key Architectural Features

* **Data-Driven Dynamic Carousels:** Upcoming and Past Productions are dynamically segregated, structured, and injected into distinct independent horizontal view rails via a centralized JavaScript array manager.
* **Independent Sliding Slider Tracks:** The *Stage Design* and *Scenography* modules leverage completely isolated directional slider controls, allowing fluid touch-and-click traversal with zero side-effect interference across sections.
* **Asynchronous Form Processing:** Intercepts contact requests natively utilizing the `Fetch API` to pipe structural payload data securely back through the Web3Forms processing server without causing page reloads.
* **Infinite Corporate Logo Marquee:** An infinite sliding loop carousel showcasing partner brands that automatically freezes its cinematic timeline coordinates on pointer hover to allow seamless outbound link traversal.
* **Native Security Implementations:** Integrated anti-spam Honeypot checks (`botcheck`) preventing system script manipulation from background scrapers or email harvesters.

---

## 🛠️ Tech Stack & Compliance

* **Markup:** Semantic HTML5.
* **Styles:** Pure, adaptive CSS3 driven by responsive CSS Grid modules, Flexbox configurations, GPU-accelerated transition transforms, and layout fluid `clamp()` variables.
* **Scripts:** Vanilla ECMAScript 6 (Native DOM APIs, Dynamic Mapping Arrays, Asynchronous Fetch REST transactions).
* **Legal Compliance:** Outlines fully transparent data management parameters compliant under the Florida Information Protection Act (FIPA) regulations.
